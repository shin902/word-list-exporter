# WordList

フラッシュカード形式で学習できるWebアプリケーション

## 概要

WordListは、問題と解答のペアを登録・管理し、ランダムに出題されるフラッシュカード形式で効率的に学習できるWebアプリケーションです。ブラウザ上で動作し、データはローカルストレージに保存されます。

## 主な機能

### 単語カード管理
- 問題と解答のペアを作成、編集、削除
- カテゴリによる分類管理
- カテゴリ別のグルーピング表示

### 学習機能
- ランダム出題による効率的な学習
- 問題表示 → 解答確認 → 次の問題へのシンプルなフロー
- 学習セッションの完了追跡

### テキスト装飾
- 上付き文字サポート（例: x^2 → x²）
- 下付き文字サポート（例: H_2O → H₂O）
- 数式や化学式の表現に対応

### データ永続化
- ブラウザのローカルストレージを使用
- オフラインでの利用が可能
- データのインポート/エクスポート（拡張機能）

## 画面構成

### ホーム画面
- 学習開始
- 単語カード一覧表示

### 学習画面
- カテゴリ表示
- 問題・解答の表示
- シャッフル機能

### 一覧画面
- カテゴリ別の表示
- 編集・削除機能
- 並び替え機能

### 追加画面
- 新規カードの作成
- カテゴリの選択・追加

## 技術スタック

- HTML/CSS/JavaScript（Vanilla JS）
- ローカルストレージ（データ永続化）
- レスポンシブデザイン（モバイル対応）

## データモデル

各単語カードは以下の構造を持ちます：

```javascript
{
  category: String,  // カテゴリ名
  question: String,  // 問題文
  answer: String     // 解答文
}
```

## 使い方

1. 単語カードを追加（問題、解答、カテゴリを入力）
2. ホーム画面から「学習開始」をクリック
3. 問題が表示されたら考えて「答えを表示」をクリック
4. 解答を確認して「次へ」で次の問題へ
5. すべての問題を終えたら完了画面が表示されます

## 拡張機能（予定）

- 学習履歴・統計の記録
- 復習スケジューリング（間隔反復法）
- データのインポート/エクスポート（JSON, CSV）
- 複数デッキの管理
- ダークモード対応
